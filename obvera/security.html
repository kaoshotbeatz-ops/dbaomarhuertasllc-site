Architecture & isolation

How customer data is kept separate, secrets stay opaque, and one tenant can't reach another.

Per-tenant database

Each customer gets a dedicated MariaDB database on AWS RDS. Tables are not shared between tenants. AsyncLocalStorage binds the DB to the request — handlers don't have to think about it.

Per-tenant vault

Customer-provided secrets (Cloudflare, Slack, Jira API tokens, etc) live in that tenant's envelope-encrypted vault. The platform owner cannot read them.

Domain separation

Sub-orgs inside one tenant for parent + subsidiary, multi-BU, MSP under one install. Hierarchical via parent_domain. ServiceNow's domain separation, ported.

Tenant signup → portal

company.dbaomarhuertasllc.com gives a customer their own portal in 30 seconds. No cross-tenant routing. No noisy-neighbor query interference.

Auditability & non-repudiation

Every privileged action is hash-chained. Any deletion, insertion, or content edit breaks the chain — and the verifier shows you exactly which entry.

Truth Ledger (hash-chained)

Every entry includes its predecessor's SHA-256. GET /ledger-verify walks the whole chain and returns the first broken link. Compliance-grade non-repudiation.

Per-tenant audit log

Standard audit_log captures every API call — actor, action, detail, source. Mapped to PCI 10.1 & HIPAA §164.312(b).

Delete guard + restore

Delete-guard business rules snapshot records before deletion. Restore from snapshot if needed.

Field-level masking

Sensitive fields (push_tokens, unlock_tokens, vault secrets) are ACL-gated; non-admins see ⟦restricted⟧ sentinel.

Access control (ServiceNow-grade)

Roles × Groups × Personas, with an ACL inspector that explains every allow/deny.

124 roles available

Each module exposes 4 tiers (admin / dev / user / viewer) for fine-grained access. Plus global roles for platform-wide operations.

12 seeded personas

CIO, CISO, IT Service Desk Manager, Help Desk Agent, IT Asset Manager, Procurement Mgr, Finance Mgr, HR Manager, Sales Mgr, Compliance Officer, DevOps Engineer, Developer.

Groups · members · roles

Same shape as ServiceNow sys_user_grmember + sys_group_has_role. Users inherit roles transitively.

ACL inspector

One call: "WHY can/can't user X do op Y on table Z?" Returns direct roles + roles via every group + roles via every persona + final verdict.

Privileged Access (PAM)

Delinea-class. Time-boxed grants, approval workflow, auto-expiry sweeper. Every grant + use ledger-chained.

MFA / TOTP

TOTP via authenticator app (Google Authenticator / 1Password / Authy). Per-user enrollment, admin-enforced.

Resilience & ops

High availability across AWS Availability Zones. Auto-rollback on failed deploy. Backups tracked by freshness.

HA across AZs

Production EC2 in us-east-1a + standby in us-east-1b. Both serve same code, same RDS. Either box dies, the other carries.

Fleet promote w/ auto-rollback

One command ships to both boxes. Per-box health gate. Auto-rollback if /healthz fails post-restart. Auto-creates per-tenant change records.

Backups

RDS automated backups + per-tenant exports + delete-guard snapshots + MariaDB dumps. Tracked with file count, total MB, freshness (> 30h = red).

Service map

15-lane interactive SVG of the whole estate: edge, AWS, on-prem, identity, virt, monitoring, staging, tenants, integrations, modules, grants, sensors. Click any node to drill.

Cost & inventory awareness

cp_infra table self-discovers every EC2, RDS, Cloudflare zone, on-prem host. Monthly cost rolls up.

Observability

/metrics Prometheus endpoint (counter + histogram + gauge). Structured request logs. Truth Ledger for security events.
